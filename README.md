# 🎨 Interactive Web Page

## 📌 Project Description

This project is a simple interactive web page built using **HTML, CSS, and JavaScript**.
It demonstrates basic DOM manipulation and event handling.

The page includes:

* A greeting header
* Color boxes (Red, Blue, Green, Yellow)
* Input field to enter a user's name
* Button to display a personalized greeting

---

## 🚀 Features

* 🧑‍💻 Dynamic greeting message
* 🎨 Clickable color boxes that change background color
* ⚡ Real-time DOM manipulation using JavaScript
* ✨ Clean and responsive UI design

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## 🎯 Functionality

### 1. Greeting Feature

* Enter your name in the input field
* Click the **Greet** button
* The header updates to:

```
Hello, [Your Name]
```

---

### 2. Color Boxes

* Click on any box:

  * The box background changes to its respective color
  * Text color adjusts for visibility
    (e.g., Yellow → Black text)
